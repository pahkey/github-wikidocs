깃허브와 위키독스 연동에 대한 학습을 **마무리**합니다.

## 배운 내용 정리

이 책을 통해 다음 내용을 학습했습니다.

### 01. 소개

- 위키독스가 무엇인지 이해했습니다.
- 깃허브 연동의 개념을 배웠습니다.

### 02. 연동의 장점

- Git 버전 관리의 강력함을 확인했습니다.
- 협업 기능(PR, 리뷰, 이슈)을 알아보았습니다.

### 03. 워크플로우

- 기본 작업 흐름을 익혔습니다.
- 브랜치 전략을 학습했습니다.

### 04. 실전 활용

- 기술 문서 작성 방법을 배웠습니다.
- 팀 협업 사례를 살펴보았습니다.

## 핵심 요약

| 항목 | 핵심 내용 |
|------|----------|
| 연동 | 깃허브 → 위키독스 자동 동기화 |
| 장점 | 버전 관리, 협업, 자동화 |
| 워크플로우 | pull → 작성 → commit → push |
| 활용 | 기술 문서, 팀 협업 |
