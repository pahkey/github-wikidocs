# 02. 연동의 장점

깃허브와 위키독스를 연동하면 문서 관리의 효율성이 크게 향상됩니다.

## 주요 장점 개요

깃허브 연동이 제공하는 핵심 장점은 다음과 같습니다.

### 버전 관리의 힘

깃(Git)의 강력한 버전 관리 기능을 문서에 적용할 수 있습니다. 모든 변경 이력이 기록되어 언제든 이전 버전으로 돌아갈 수 있습니다.

### 협업의 용이성

여러 사람이 동시에 문서를 작성하고 검토할 수 있습니다. Pull Request를 통한 리뷰 프로세스로 문서 품질을 높일 수 있습니다.

### 자동화된 워크플로우

push 한 번으로 위키독스에 자동 반영됩니다. 수동으로 복사/붙여넣기할 필요가 없어 실수를 줄일 수 있습니다.

## 기존 방식과의 비교

| 항목 | 웹 에디터 방식 | 깃허브 연동 방식 |
|------|---------------|-----------------|
| 버전 관리 | 제한적 | Git 전체 기능 |
| 협업 | 어려움 | PR/리뷰 지원 |
| 백업 | 수동 | 자동 (Git) |
| 오프라인 작업 | 불가능 | 가능 |
| 에디터 선택 | 웹 에디터만 | 자유 선택 |

- **02-2. 협업 기능**: Pull Request, 코드 리뷰, 이슈 관리
