**위키독스(WikiDocs)**는 누구나 책을 만들고 공유할 수 있는 온라인 문서 플랫폼입니다.

## 위키독스의 특징

### 무료 출판 플랫폼

위키독스는 별도의 비용 없이 책을 작성하고 출판할 수 있습니다. 작성한 내용은 즉시 온라인에 공개되어 누구나 읽을 수 있습니다.

### 마크다운 지원

마크다운(Markdown) 문법을 지원하여 간단하게 서식이 있는 문서를 작성할 수 있습니다.

```markdown
# 제목
## 소제목
- 목록 항목
**굵은 글씨**
```

### 구조화된 목차

책의 구조를 목차(Table of Contents)로 관리하여 체계적인 문서 구성이 가능합니다. 독자는 목차를 통해 원하는 내용으로 쉽게 이동할 수 있습니다.

## 위키독스의 활용 분야

위키독스는 다양한 분야에서 활용되고 있습니다.

- **프로그래밍 튜토리얼**: Python, JavaScript, Java 등의 학습 자료
- **기술 문서**: API 문서, 사용자 가이드, 개발 매뉴얼
- **교육 자료**: 강의 노트, 학습 교재
- **개인 지식 정리**: 학습 내용 기록, 노하우 공유

## 왜 위키독스인가?

| 장점 | 설명 |
|------|------|
| 접근성 | 웹 브라우저만 있으면 어디서나 접근 가능 |
| 편의성 | 복잡한 도구 없이 마크다운으로 작성 |
| 공유 | URL 하나로 누구와도 쉽게 공유 |
| 커뮤니티 | 활발한 한국어 기술 문서 커뮤니티 |
