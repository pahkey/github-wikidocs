## 대상 독자

- 기술 문서를 체계적으로 관리하고 싶은 개발자
- 팀 프로젝트의 문서화를 개선하고 싶은 팀 리더
- 깃허브를 활용한 문서 관리에 관심 있는 분
- 위키독스에서 책을 출판하고자 하는 저자

## 핵심 장점

- **버전 관리**: Git의 강력한 이력 관리 기능 활용
- **협업**: Pull Request와 코드 리뷰를 통한 팀 협업
- **자동화**: push 한 번으로 위키독스에 자동 반영
- **유연성**: 익숙한 에디터와 도구 사용 가능
